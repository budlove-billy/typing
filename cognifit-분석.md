# CogniFit 게임 분석 & Mallow 추가 게임 제안 (2026-07-10)

> 출처: [CogniFit 두뇌 게임](https://www.cognifit.com/kr/public/games) (60+종, 5분류: 기억·협동·지각·주의·추론)
> 비교 대상: Mallow 활성 13종.

## 1. 카테고리 커버리지 비교

| CogniFit 축 | CogniFit 예시 | Mallow 현황 |
|---|---|---|
| 기억 | 수련, 사탕 공장, 수학적 혼돈 | ✅ 4종 (순간기억·인원수·엔백·카드) |
| 주의(반응·억제·분할) | 개구리 모험, 컬러 러쉬, 교차로 | ✅ 3종 (스트룹·순서잇기·반응속도) |
| 추론(계획·유연성) | 캔디 라인업, Robo Factory, 스도쿠 | ✅ 4종 (규칙·IQ·스도쿠·암산) |
| 지각(공간) | 퍼즐, 3D 벽돌깨기 | ✅ 1종 (도형 회전) |
| **협동(눈-손 협응)** | 두더지 잡기, 나비 사냥, 벽돌깨기, 폭탄 테니스 | ❌ **완전 공백** |
| **청각(지각·기억)** | 미쳐 버리겠네(사이먼류), 멜로디 메이헴, 멜로디 테니스 | ❌ **완전 공백** |
| 시각 탐색 | 애완동물 찾기, 단어 찾기 | ❌ 공백 (기존 백로그 '다른 것 찾기'와 동일) |

CogniFit의 강점 = **축의 폭(청각·협응)** + AI 적응 난이도. 우리 적응형 난이도는 이미 게임별 구현되어 있으므로, 격차는 **청각·협응 2개 축**이 핵심.

## 2. 추가 게임 제안 (우선순위)

1. **★ 두더지 잡기 (말로우 팡팡)** — 협응+반응+억제 축 신설.
   구멍에서 튀어나오는 **마스코트(말로우)를 탭**, 가짜(빨간 것)는 참기 → Go/No-Go 확장판.
   캐주얼 대중성 최고 + **마스코트 활용 첫 게임**(브랜드 시너지). 구현: 3×3 그리드 탭, 기존 패턴 재사용, 1일.
2. **★ 멜로디 기억 (사이먼류)** — 청각 축 신설.
   색+음이 순서대로 재생 → 따라 누르기, 매판 +1 길어짐. **방금 구축한 WebAudio 재사용**(4음 스케일).
   전통 게임(Simon, 1978) 퍼블릭 컨셉. 구현: 4버튼 패드, 1일.
3. **다른 것 찾기 (시각 탐색)** — 기존 백로그(Lumosity Star Search=CogniFit 애완동물 찾기). 비슷한 이모지 중 하나 다른 것 탭.
4. 어림 계산(Top That류) / 미로 경로 계획(펭귄 탐험류) — 후순위(Phase B/C 데이터 보고).

→ 1·2번을 넣으면 **인지 축 8개 전부 커버**(기억·주의·유연·공간·계산·논리·협응·청각), CogniFit 대비 "축 공백" 논리가 사라짐.

## 3. 차별화 재확인
- CogniFit도 결국 가입+구독 장벽 → 우리 "가입 없이 3초" 포지션 그대로 유효.
- CogniFit 카피 톤 "하루 10분 뇌 건강 유지"는 효능 암시 — 우리는 재미/휴식 톤 유지(FTC 원칙).
