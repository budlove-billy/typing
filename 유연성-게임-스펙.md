# 유연성 축 + 홈/공유 일반화 스펙

> 대상: `index.html`(= `brain_app.html`) 단일 HTML SPA "말랑말랑 두뇌체조"
> 이 문서는 한 배치에서 함께 넣은 세 가지를 기록: **규칙 바꾸기(유연성 축)**, **홈/대시보드**, **공유카드 일반화**.

---

## 1. 규칙 바꾸기 (Rule Switch) — `id: switch` · 유연성 축

### 게임성
규칙이 매 시행 확률적으로 바뀐다. 지금 규칙에 맞춰 숫자를 판단 → **인지 전환(set-shifting)** 훈련.
- 규칙 A(parity): 숫자가 **홀수/짝수**?
- 규칙 B(magnitude): 숫자가 **5보다 작다/크다**?
- 숫자는 `SW_NUMS=[1,2,3,4,6,7,8,9]`(5 제외 → 크기 규칙 애매 방지).

### 규칙
- 매 시행: 확률 `switchP`로 규칙 토글 → 새 숫자 → 두 버튼 라벨을 현재 규칙에 맞게 교체.
  - parity: 버튼 `홀수 / 짝수`(dataset.k = odd/even)
  - magnitude: 버튼 `5보다 작다 / 크다`(low/high)
- 정답 combo +점수(`10+(combo-1)*2`), 오답 combo=0 + 시간 −2초 + 버튼 플래시.
- 60초 타임어택. 난이도 = 전환 빈도: easy 0.3 / normal 0.45 / hard 0.6(자주 전환 = 전환비용↑ = 어려움).
- 결과: 점수 · 정답 수 · 최고 콤보 · 최고 기록. localStorage `brain.switch.best`.
- 상태 `SW`. 함수: `setSwitchDiff` `startSwitch` `swNextTrial(first)` `swAnswer(btn)` `finishSwitch`.
- 규칙 배너는 색으로 구분(parity=보라, magnitude=초록)해 전환을 시각적으로 인지.

## 2. 홈 / 대시보드 — `id: home` (기본 랜딩)

- 앱 진입 시 첫 화면(`.screen.active`). nav 맨 앞 홈 탭.
- **히어로**: 인사말 + 오늘의 동기부여 + **연속 출석** 뱃지(🔥 N일 연속).
- **전체 트레이닝 그리드**: `HOME_GAMES` 매니페스트로 11개 바로가기 카드(이모지+이름).
  점수형 게임(flash/count/stroop/trail/switch)은 카드에 최고기록 뱃지(🏆 = 난이도별 all 최댓값).
- **연속 출석**: `homeVisit()` → localStorage `brain.visit`{last, streak, total}.
  - 어제(`bt_yesterday()`) 방문이면 streak+1, 하루라도 공백이면 1로 리셋, 같은 날 재방문은 불변.
- `renderHome()`는 `showScreen('home')`·INIT·언어 전환(현재 홈일 때) 시 호출.
- 인원수 세기 nav 아이콘은 🏠→👥로 변경(홈과 시각적 구분).

## 3. 공유카드 일반화 (전 게임 공용)

- `makeShareCard({emoji, title, subtitle, lines[], desc, note, from, to})` → 600×800 canvas 반환.
  그라데이션(from→to) 배경 + 로고 + 이모지 + 제목 + 점수 라인들(+선택적 desc/note) + 워터마크.
- `shareOrDownload(canvas, filename, shareText)`: `navigator.canShare({files})` → Web Share, 아니면 PNG 다운로드.
- `gameShare('<id>')`: 각 게임 상태에서 카드 구성(게임별 강조색) → 공유. flash/count/stroop/trail/switch 결과에 버튼.
- 두뇌 유형 결과도 동일 헬퍼로 통일(`_btCard` 미리보기 + `btShare`). 기존 `btDrawCard`/`#bt-canvas` 제거.

## 4. 검증
- `node --check`(JS 문법), div 균형(351/351), 화면·nav 12종 일치, i18n 전수 존재(누락 0).
- 규칙 바꾸기: parity/magnitude 정답 판정 8숫자 전수 검증, 5 제외 확인, 전환율 난이도 비례(0.30<0.60).
- 홈 출석: 첫방문/연속/동일일/공백리셋 4케이스 검증.
