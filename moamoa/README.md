# 모아모아 — 오늘의 낱말 묶기 (한국어 Connections류)

> 근거 분석: [../데일리퍼즐-포맷-분석.md](../데일리퍼즐-포맷-분석.md)
> 포지션: NYT Connections(연 33억 플레이, 국내 정착작 없음) 포맷의 한국어 선점작.
> 법적 설계: 규칙(아이디어)만 차용, 표현은 전부 독자 제작 — 이름·UI·색·데이터 원작과 무관. 워들류 아님(NYT DMCA 계열 회피).

## 게임 규칙

- 16개 낱말을 **공통점이 있는 4개 그룹(각 4낱말)** 으로 묶는다.
- 4개 선택 → "확인" → 맞으면 그룹 공개, 틀리면 목숨 1 감소 (총 4번 틀리면 종료).
- 3개가 같은 그룹이면 **"하나만 달라요!"** 힌트.
- 난이도 4색: 🟨 쉬움 → 🟩 보통 → 🟦 어려움 → 🟪 함정(교차 함정 포함).
- **하루 한 판, 모두 같은 문제, KST 자정 리셋.**

## 성공 공식 구현 (꼬맨틀·꼬들 검증)

| 요소 | 구현 |
|---|---|
| 하루 한 판 | KST 날짜 기준 퍼즐 인덱스, 완료 후 내일 카운트다운 |
| 이모지 공유 | 추측 내역을 🟨🟩🟦🟪 그리드로 클립보드/공유 |
| 로그인 없음 | 진행·기록 전부 localStorage |
| streak | 연속 성공 일수 + 최대 기록 |
| 모바일 우선 | 터치 타깃 크게, 480px 최적화, 3초 내 시작 |

## 콘텐츠 제작 원칙 (퍼즐 데이터)

1. 그룹당 4낱말 × 4그룹 = 16낱말, 퍼즐 내 중복 금지.
2. **교차 함정 필수**: 최소 1개 낱말은 다른 그룹처럼 보이게 (예: `훈수`가 "○수 직업"처럼 보이지만 정답은 "바둑 용어").
3. 한국어 특화 카테고리 우선: 속담·한자성어·신조어·K콘텐츠·생활문화 — 원작이 못 하는 로컬 재미.
4. 노랑은 초등학생도, 보라는 어른도 갸웃하게 — 난이도 스펙트럼이 공유 욕구를 만든다.
5. 제작 파이프라인: LLM 초안 생성 → 사람 검수(중의성·함정 품질) → `PUZZLES` 배열 추가.

## 글로벌(다국어) 검토 결론 — 2026-07-10

**핵심: Connections류는 "번역"이 아니라 언어별 콘텐츠 재창작이다.** 낱말·함정·문화 레퍼런스가 게임의 전부라, UI 번역만으로는 게임이 성립하지 않음.

| 시장 | 경쟁 상태 (조사 실측) | 판단 |
|---|---|---|
| **한국어** | Connections 정착작 없음 | **주력. 선점 유효** |
| 영어 | **극포화** — connectionsgame.org, Connections Unlimited(퍼즐 500+, 서·포어판까지), Connections+, dailycryptic, 스포츠 에디션 등 무한판·아카이브·커스텀 제작기가 전부 무료 경쟁 중. NYT 원작(연 33억 플레이)도 무료 | **진입 ✗** — 신규 진입자가 얻을 자리 없음 |
| 태국어 | 워들류(Thwordle, 국립코퍼스 기반 word.in.th, tower.in.th 퍼즐 모음)는 활발하나 **Connections류 부재** | **후보 ○, 단 지금은 보류** — 태국어 낱말·문화 검수 역량 없이는 품질 보장 불가. 한국어판 검증 후 현지 검수자 확보 시 진행 |

**아키텍처 결정**: 지금 만드는 코드는 언어 중립로 유지(UI 문자열·퍼즐 데이터 분리 가능 구조). 시장별 진출은 "한국어 검증 → 태국어(검수자 확보 시) → 영어는 영구 제외" 순.
