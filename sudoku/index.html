이런 점이 좋아요

  • 가입·설치 없음 — 브라우저에서 바로 시작, 광고도 없습니다.
  • 난이도 3단계 — 쉬움·보통·어려움, 매판 새 퍼즐 생성.
  • 편의 기능 — 노트(메모), 힌트 3회, 되돌리기, 실수 3회 체크.
  • 모바일 최적화 — 큰 숫자와 눈이 편한 배경으로 폰에서도 편하게.
  • 기록 저장 — 최고 기록이 내 기기에 저장돼 다음에 이어서 도전.

스도쿠 하는 법

9×9 격자를 1~9 숫자로 채우는 퍼즐입니다. 규칙은 하나 — 각 행, 각 열, 각 3×3 박스에 같은 숫자가 두 번 나오면 안 됩니다. 확실한 칸부터 채우고, 헷갈리는 칸은 노트 기능으로 후보 숫자를 적어두세요. 실수가 3회 쌓이면 게임이 끝나니 신중하게!
