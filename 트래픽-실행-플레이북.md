# 트래픽 유입 실행 플레이북 (playmallow.com)

> 2026-07-21. 신생 도메인(출시 11일)용. **빠른 채널 → 느린 채널** 순. 코드 SEO는 제가 처리, 아래 ★는 사용자 직접 실행 필요.

## 이번 주 (색인 = 모든 것의 전제)
신생 사이트는 "검색에 안 뜨는" 게 아니라 "아직 색인 안 된" 상태. 먼저 구글·네이버에 존재를 알린다.

1. ★ **구글 서치콘솔** (search.google.com/search-console)
   - 이미 등록돼 있으면: `sitemap.xml` 제출 확인 → **URL 검사**로 `/`, `/unse/`, `/zodiac/`, `/tarot/`, `/ttirank/`, `/privacy/` 각각 "색인 생성 요청".
   - 미등록이면: 도메인 속성 추가(가비아 DNS TXT 인증, kimpwatch 방식과 동일).
2. ★ **네이버 서치어드바이저** (searchadvisor.naver.com)
   - 사이트 등록 → 소유확인 → `sitemap.xml`·`robots.txt` 제출 → **웹페이지 수집 요청**으로 운세 페이지 등록.
   - 네이버는 국내 검색 점유율 큼 + 운세 검색 수요가 네이버에 몰림. **KR 트래픽 핵심.**
3. ★ **빙 웹마스터**(bing.com/webmasters) — 서치콘솔에서 사이트맵 가져오기(import) 한 번이면 됨. 저비용 보너스.

## 이번 주~다음 주 (가장 빠른 실제 유입: 커뮤니티 시딩)
검색 랭킹은 몇 주 걸리지만, **공유 콘텐츠는 오늘 클릭을 만든다.** 운세 결과가 미끼.

4. ★ **카카오톡 오픈채팅 / 네이버 카페 / 커뮤니티** 에 운세 결과 자연 공유
   - 타깃: 맘카페, 학부모, 여성·MZ 커뮤니티(더쿠·인스티즈), 지인 단톡방.
   - 방식: 스팸 금지. "오늘 운세 봤는데 소름" 식으로 **결과 이미지 + 링크** 하나. 콘텐츠가 퍼지게.
   - 복붙 예시:
     > 오늘 띠 순위 이거 은근 맞아서 공유 🐰 [내 띠 몇 위인지 보기] https://playmallow.com/ttirank/
     > 무료 타로 오늘의 카드 뽑아봤는데 오늘 딱 맞음… https://playmallow.com/tarot/
5. ★ **당근 동네생활 / 에브리타임** — 두뇌게임·운세 둘 다 반응층. 지역/대학 커뮤니티.

## 2~4주 (복리 채널: 콘텐츠 백링크)
6. ★ **네이버 블로그 1개 개설** → 주 1~2글. 네이버 검색 노출에 유리 + 본사이트 백링크.
   - 첫 글 제목 후보(검색 수요 큰 롱테일):
     - "무료 오늘의 운세 사이트 추천 (가입 없이 바로)"
     - "무료 타로 오늘의 카드 뽑는 법"
     - "설치 없이 하는 무료 두뇌게임 33選"
   - 각 글 하단에 playmallow.com 관련 페이지 링크.
7. ★ **글로벌 게임 포털** (두뇌게임 한정, 영어권 무료 트래픽)
   - itch.io(가장 쉬움, 단일 HTML 업로드) → CrazyGames·GameDistribution 순.
   - Reddit r/webgames, r/WebGames에 Show 글 1회.

## 계절 스파이크 (미리 준비 = 선점)
8. **신년운세 / 설·추석 / 수능** 전용 랜딩을 시즌 2~3주 전 미리 제작(포스텔러가 매년 먹는 트래픽).
   - 예: `/newyear/` 2026 신년운세, `/suneung/` 수능 운세. → 제게 "시즌 랜딩 만들어줘" 하시면 제작.
