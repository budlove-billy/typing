이런 점이 좋아요

  • 가입·설치 없음 — 브라우저에서 바로, 아이도 어른도 쉽게.
  • 60초 타임어택 — 짧고 굵게, 자투리 시간에 딱.
  • 점점 어려워짐 — 판을 다 맞추면 카드 쌍이 늘어 난이도가 올라가요.
  • 콤보 보너스 — 연속으로 맞히면 점수가 더 크게 붙습니다.
  • 기록 저장 — 최고 점수가 내 기기에 저장돼 다음에 도전.

카드 짝 맞추기 하는 법

모든 카드는 뒤집힌 상태로 시작합니다. 카드를 두 장 뒤집어 그림이 같으면 짝 성공(그대로 사라지거나 표시됨), 다르면 다시 뒤집힙니다. 이때 어떤 그림이 어디 있었는지 기억해 두는 게 핵심이에요. 판의 모든 짝을 맞추면 다음 판은 카드 쌍이 늘어 더 어려워집니다. 60초 안에 최대한 많은 짝을 맞춰 점수를 올려 보세요.

시각 작업기억(본 것을 잠시 기억하는 능력)을 가볍게 쓰는 대표적인 게임입니다.
