# 구글 애드센스 신청 가이드 (playmallow.com)

> 2026-07-21 작성. 신규 신청 기준. 심사는 보통 **数일~4주**, 신생 도메인은 트래픽이 있어야 통과율↑.

## 지금 상태 (준비 완료 항목)
- ✅ **개인정보처리방침** `/privacy/` — 쿠키·GA·AdSense 제3자 쿠키 고지 + opt-out 링크(승인 필수 요건)
- ✅ **오리지널 콘텐츠** — 게임 33종 + 운세 + 가이드/랜딩 (양산형 아님)
- ✅ **정책 안전 설계** — 게임 플레이 화면엔 광고 안 붙임(오클릭=계정 정지 위험 회피)
- ⏳ 트래픽 — 신생 도메인이라 심사 리스크. 트래픽 작업 병행 필요.

## 신청 절차 (사용자가 직접, 약 10분)
1. **adsense.google.com** 접속 → 구글 계정 로그인.
2. 사이트 URL에 **playmallow.com** 입력, 국가=대한민국, 지급 통화 선택.
3. 애드센스가 주는 **연결 코드(`<script ... data-ad-client="ca-pub-XXXXXXXXXXXXXXXX">`)** 확인.
   → **이 `ca-pub-...` 값을 저(클로드)에게 알려주시면** 사이트 전 페이지 `<head>`에 삽입 + `ads.txt` 생성까지 제가 처리합니다.
4. 코드 삽입·배포 후 애드센스 대시보드에서 **"검토 요청"**.
5. 이메일로 승인/보완 통보 대기.

## 승인 후 제가 자동으로 할 일 (pub-id 받으면)
1. **연결 스니펫**을 `index.html`(=brain_app.html) + 정적 랜딩 17개 `<head>`에 삽입.
2. **`ads.txt`** 루트에 생성:
   ```
   google.com, pub-XXXXXXXXXXXXXXXX, DIRECT, f08c47fec0942fa0
   ```
3. **광고 슬롯 배치**(정책 안전 위치만):
   - 정적 랜딩/가이드 페이지 본문 하단
   - 운세 **결과 화면**(총운 확인 후 자연스러운 지점)
   - 게임 **결과 카드** 화면(플레이 중 아님)
   - 홈·내기록 화면 하단 배너
   - ❌ 게임 플레이 화면, 버튼 근처, 팝업 위 = 절대 배치 안 함
4. "광고 없습니다" 카피 수정(2048/·sudoku/·water-sort/ 랜딩 3곳).

## 반려되면 (흔한 사유·대응)
- **"가치가 낮은 콘텐츠(low value content)"** → 트래픽·색인 페이지 부족. 2~4주 트래픽 쌓고 재신청.
- **"사이트 미가동/탐색 불가"** → 색인 안 됨. 서치콘솔 색인 요청으로 해결.
- 반려는 재신청 가능. 좌절 금지 — 신생 사이트 흔한 통과 의례.

## 수익 현실 (기대치 세팅)
- 한국 캐주얼/운세 애드센스 RPM 대략 **1,000뷰당 1,000~5,000원**.
- 즉 월 3만 PV ≈ 월 3~15만 원 수준(변동 큼). **트래픽이 곧 수익.**
