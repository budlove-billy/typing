# Lumosity 분석 & 우리 앱 적용 제안

> 목적: Lumosity가 "더 전문적으로 느껴지는" 이유를 게임·서비스 구조 양면에서 분석하고,
> 우리 "말랑말랑 두뇌체조"(게임 15종)에 참고할 점을 정리.
> 출처: [Lumosity — Wikipedia](https://en.wikipedia.org/wiki/Lumosity), [Lumosity Brain Games](https://www.lumosity.com/en/brain-games/).

---

## 0. 핵심 통찰 — 차이는 "게임"이 아니라 "구조"에 있다

우리 게임 15종은 Lumosity의 개별 게임과 **인지적으로 동급**이다(아래 대응표). 그런데도 Lumosity가
더 전문적으로 느껴지는 이유는 게임을 감싸는 **메타 구조** 때문이다:

1. **Fit Test(진단)** — 처음에 몇 개 게임으로 기준선을 측정, "당신의 현재 수준"을 숫자로 제시.
2. **Daily Workout(오늘의 훈련)** — 매일 **엄선된 3~5개 게임을 골라주고 순서대로** 하게 함(=미션).
   "오늘 뭐하지" 고민 제거 + 매일 다른 조합으로 재방문 유도.
3. **LPI(Lumosity Performance Index)** — 게임 점수를 **인지 영역별 정규화 지수**로 환산.
   영역별(기억/주의/속도/유연성/문제해결) 점수와 **시간에 따른 성장 그래프** 제공.
4. **적응형 난이도 + 튜토리얼 + 습관 추적**(진행 그래프로 꾸준함 유도).

→ 즉 "게임 모음"이 아니라 **측정 → 처방(오늘의 미션) → 성장 추적**의 루프. 이게 전문적 느낌의 정체.
사용자가 말한 **듀오링고식 단계/미션**이 바로 이 "처방(Daily Workout)" 레이어에 해당.

## 1. 게임 대응표 — 우리는 이미 대부분 커버한다

| Lumosity 영역 | Lumosity 대표 게임 | 우리 대응 게임 | 상태 |
|---|---|---|---|
| 기억 | Memory Match(카드), Memory Matrix(격자) | 카드 짝, 순간기억, 엔백, 인원수 | ✅ 오히려 더 많음 |
| 속도 | Speed Match, Highway Hazards | 반응 속도, 순서 잇기 | ✅ |
| 주의 | **Lost in Migration(플랭커)**, **Star Search(시각탐색)** | (순서 잇기 일부) | ⚠️ **플랭커·시각탐색 없음** |
| 유연성 | Brain Shift(규칙전환), Color Match(스트룹) | 규칙 바꾸기, 색깔 맞추기 | ✅ |
| 문제해결 | **Pet Detective(경로계획)**, Masterpiece(공간조립) | IQ, 스도쿠, (도형 회전) | ⚠️ **계획/경로 없음** |
| 언어 | **Word Bubbles(낱말생성)**, Word Snatchers(애너그램) | 영단어(학습), 한글 타자 | ⚠️ **낱말놀이 없음** |
| 수리 | Chalkboard, **Raindrops(낙하 연산)**, **Top That(어림)** | 암산 | ⚠️ **어림/추정 없음** |

**결론:** 게임 갯수·다양성은 우리가 밀리지 않는다. 비어 있는 소수 영역만 보강하면 됨.

## 2. 참고할 만한 신규 게임 후보 (우리에 없는 것, 웹 구현 쉬운 순)

1. **플랭커 — "화살표 방향"(Lost in Migration)** ★추천
   가운데 화살표 방향을 답하되 양옆 방해 화살표(→←→) 무시. 주의·억제. 탭 4방향. 구현 쉬움.
2. **시각 탐색 — "다른 것 찾기"(Star Search)** ★추천
   비슷한 도형들 사이에서 딱 하나 다른 것 탭. 주의·시지각. 격자 렌더. 구현 쉬움.
3. **어림 계산 — "많은 쪽"(Top That)** 
   두 묶음 중 값이 큰 쪽을 순간 어림으로 선택. 수 감각(number sense). 우리 암산과 다른 축.
4. **낱말 만들기 — 애너그램/한글 자모 조합(Word Bubbles)**
   주어진 글자로 단어 만들기(한국어 낱말 or 영어). 언어. 사전 데이터 필요(중간 난이도).
5. **낙하 연산 — Raindrops** 
   위에서 떨어지는 식을 바닥 닿기 전 풀기. 압박형 암산. 우리 암산의 변형.
6. **경로 계획 — Pet Detective** (난이도 높음)
   여러 지점을 최단으로 도는 순서 짜기(외판원 축소판). 계획/실행기능. 구현 부담 큼 → 후순위.
7. **작업기억 갱신 — Tidal Treasures**
   이미 고른 적 없는 아이템만 계속 고르기(집합이 커짐). 순간기억과 다른 갱신형 기억.

**추천 우선순위:** 플랭커 → 시각탐색 → 어림 (셋 다 웹 구현 쉽고 비어 있는 축을 정확히 메움).

## 3. 서비스 구조 제안 — "전문적 느낌"을 만드는 메타 레이어

사용자 질문(듀오링고식 단계 / 자유 / 미션)에 대한 답: **셋을 배타적으로 고르지 말고 층으로 쌓자.**

### A. 자유 플레이 (이미 있음)
홈 그리드에서 아무 게임이나 → 유지. 캐주얼·라이트 유저용.

### B. 오늘의 미션 = Daily Workout ★가장 값짐
- 매일 **3개 게임을 축을 섞어 골라주고 순서대로** 진행(예: 기억1 → 주의1 → 유연성1).
- 날짜 시드로 결정(누구나 같은 날 같은 세트 → 공유·경쟁 가능), 다 끝내면 **오늘의 미션 완료 도장**.
- 이미 있는 연속 출석 streak와 결합 → 강력한 재방문 훅. **구현 부담 작음(기존 게임 재사용)**.

### C. 두뇌 지수 = 우리식 LPI ★전문성의 핵심
- 각 게임 점수를 **인지 축별 지수(0~1000 등)** 로 환산·누적. 홈에 **축별 레이더/막대 + 성장 추이**.
- "이번 주 기억 +40" 같은 피드백 → 측정되는 느낌 = 전문적. localStorage로 서버 없이 가능.
- 첫 실행 시 **간단 Fit Test**(축별 대표 게임 1판씩)로 기준선 잡기(선택).

### D. 코스/여정 (듀오링고식 단계) — 선택적 확장
- 축별로 "레벨 1→2→3…" 잠금 해제 경로. 몰입형 유저용. 구현 부담 중간 → B·C 다음.

**권장 순서: B(오늘의 미션) → C(두뇌 지수) → 신규 게임 1~2종 → D(코스).**
B·C만 얹어도 "게임 모음"에서 "훈련 프로그램"으로 격상된다.

## 4. ⚠️ 반드시 지킬 것 — FTC 교훈

Lumosity는 2016년 FTC와 **허위광고로 합의(벌금 최대 5천만$, 200만$로 감액)**. 문제는
"두뇌 트레이닝이 치매·노화 인지저하를 막는다"는 **과학적 근거 없는 효과 주장**이었다.
- ❌ "IQ가 오른다 / 치매 예방 / 머리가 좋아진다" 류 **효능 단정 금지**.
- ✅ "재미있게 꾸준히, 인지 자극·습관" 톤. 두뇌 유형 테스트도 **재미(entertainment)** 로 포지셔닝.
- 과학적으로도 near-transfer(그 게임을 잘하게 됨)는 있지만 far-transfer(전반적 지능 향상)는 근거 약함.

## 5. 요약 & 다음 단계

- 우리 **게임 라인업은 이미 Lumosity급**. 부족한 건 게임이 아니라 **측정·처방·성장 추적 구조**.
- 가장 값진 다음 작업: **① 오늘의 미션(Daily Workout) → ② 두뇌 지수(축별 성장 그래프)**.
- 게임을 더 넣는다면: **플랭커 / 시각탐색 / 어림** 순.
- 광고·문구는 FTC 교훈대로 효능 단정 회피.
