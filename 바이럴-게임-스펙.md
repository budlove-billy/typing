# 바이럴 축 — 두뇌 유형 테스트 + 공유카드 스펙

> 대상: `index.html`(= `brain_app.html`) 단일 HTML SPA "말랑말랑 두뇌체조"
> 목적: 두뇌훈련 자체가 아니라 **신규 유입·공유(바이럴)** 를 만드는 축.
> 연계: 결과가 앱 내 게임 추천으로 이어져 **테스트 → 게임 플레이** 유입 루프 형성.

---

## 1. 두뇌 유형 테스트 — `id: braintype`

### 흐름
intro → quiz(6문항) → result(유형 + 추천 게임 + 공유카드).

### 유형(6종) — 각 유형은 앱 내 게임으로 연결(크로스 프로모션)
| 유형 | 이모지 | 추천 게임 |
|---|---|---|
| 계산가형 calc | 🧮 | 암산(math) |
| 언어가형 word | 📚 | 영단어(vocab) |
| 기억가형 memory | 🧠 | 순간기억(flash) |
| 집중가형 focus | 🎯 | 색깔 맞추기(stroop) |
| 순발가형 quick | ⚡ | 순서 잇기(trail) |
| 추리가형 logic | 🧩 | IQ 테스트(iq) |

### 채점
- 6문항 × 4선택. 각 선택은 한 유형에 +1.
- 결과 = 최다 득점 유형. 동점은 `BT_ORDER`(정의 순) 우선.
- 문항/선택지 설계상 **6유형이 각각 정확히 4회 등장**(균형). 전 4096 조합에서 항상 유효 유형 산출(검증됨).

### 데이터/함수
- 문항·유형 텍스트는 `BT_QUESTIONS`/`BT_TYPES`에 `{ko,en,th}` 인라인(IQ 문항과 동일 방식).
  `btL(o)=o[CURRENT_LANG]||o.ko`로 현재 언어 선택. 언어 전환 시 `refreshDynamicTexts()`가 재렌더.
- 상태 `BT={idx,counts,result}`. 함수: `startBrainType` `btRenderQuestion` `btChoose`
  `btFinish`(argmax) `btShowResult` `resetBrainType`.

## 2. 결과 공유카드 (canvas) — 무료 바이럴 엔진

### 구현
- `<canvas 600×800>`(story/피드 비율)에 그라데이션 배경 + 반투명 패널 +
  로고 · 테스트명 · 유형 이모지(150px) · 유형명 · 설명(자동 줄바꿈) · 추천 게임 · 워터마크.
- `btDrawCard(type)` → `canvas.toDataURL()`로 결과 화면에 **미리보기 이미지** 즉시 표시.
- `btShare()` → `canvas.toBlob()`:
  - `navigator.canShare({files})` 지원(주로 모바일) → **Web Share**로 이미지 파일 공유.
  - 미지원(데스크톱) → **PNG 다운로드** 폴백. 아주 오래된 브라우저는 dataURL 다운로드.
- 헬퍼: `btRoundRect`(둥근 패널), `btWrapText`(설명 줄바꿈).

### 재사용 여지
- 현재는 두뇌 유형 결과에 연결. `btDrawCard`/`btShare` 패턴을 일반화하면
  스트룹·기억·IQ 점수 카드에도 확장 가능(후속 작업).

## 3. 검증
- `node --check`(JS 문법), div 균형(307/307), 화면·nav 10종 일치.
- i18n 전수 존재 확인(누락 0), 추천 대상 `nav.*`·`screen-*` 6종 존재 확인.
- 채점 로직: 유형 도달성 6/6, 커버리지 4·4·4·4·4·4, 전수 4096 조합 유효(검증됨).
- ⚠️ canvas 실제 렌더/공유는 headless 브라우저 부재로 코드 검토로만 확인(표준 Canvas API만 사용).
